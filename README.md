# Game Download Animation Demo

A high-fidelity download button animation demo following the PC233 Design System.

## Features
- Custom download button with hover animation (`@property` based).
- Flying bubble transition animation.
- Status icon transitions (Idle, Downloading, Paused, Completed).
- Elastic pop and badge increment animations.
